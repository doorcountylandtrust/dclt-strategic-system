# Website Overhaul Roadmap (2025–2026)

A high-level summary of the phases, milestones, and current status for the full DCLT website overhaul—including strategy, content, design, technical build, and launch. This roadmap complements deeper pages (Preserve Explorer, Modular Blocks, IA) and provides alignment for internal stakeholders.

---

## Phase 1: Strategy & Discovery (Q1–Q2 2025)

This phase focused on gathering internal input, aligning goals, and developing a content and architecture strategy grounded in real user and staff needs.

- [x] Staff interviews across departments
- [x] Cross-synthesis summary complete
- [x] Website audit (legacy site + priority fixes)
- [x] Homepage concept mockups explored
- [x] Sitemap and IA draft created
- [x] Supporter personas + psychographic funnels (internal planning)
- [x] Alignment with Rebrand goals and content ops

---

## Phase 2: IA & Content Modeling (Q2–Q3 2025)

Defining reusable content structures, templates, and calls to action for each major audience journey.

**Modular Content Architecture**
- [x] Content types mapped (Preserve, Story, CTA, Get Involved, etc.)
- [x] Modular content blocks documented
- [x] Field definitions scoped (ACF, GraphQL)
- [ ] Templates for each core page in progress
- [x] Project tracker system structured by milestone, task type, and phase
- [ ] Funnel and CTA mapping by persona segment in development

**Page-Level Information Architecture**
- [x] Preserve, Donate, Get Involved: wireframe layouts and block plans
- [x] Landowner “Protect Your Land” Hub structured
- [x] Peer IA scans referenced
- [ ] Homepage IA refinement underway

---

## Phase 3: Design & Wireframes (Q3–Q4 2025)

Merging rebrand with a clean, modular UI system that supports flexible storytelling, donor conversion, and preserve engagement.

**Wireframe Development**
- [x] Core wireframes (Home, Preserve, Donate, Volunteer, Contact)
- [ ] Wireframes integrated with real copy (ongoing)
- [ ] Feedback loop with staff and stakeholders

**Interactive Components**
- [x] Preserve Explorer: map-based UI prototype built
- [ ] Leaflet/React integration in test phase
- [ ] Event and stewardship overlays scoped and integrated into map planning
- [ ] Live filtering and analytics hooks planned

**Visual System Alignment**
- [x] Rebrand elements (type, color, icons) integrated
- [ ] Testing accessible color and layout patterns
- [ ] Hero and top navigation components under design
- [ ] Figma visual system exploration ongoing

---

## Phase 4: Technical Build (Q4 2025 – Q1 2026)

Developing a scalable, performance-minded site architecture using WordPress (PHP + ACF) with embedded React modules and future-friendly design.

**Dev Environment & Architecture**
- [x] Local WP theme with Tailwind and ACF initialized
- [x] Post types and custom fields (Preserve, Story, Event) structured
- [ ] React + Leaflet map components loaded via shortcode
- [ ] CMS training and fallback theme in progress

**Analytics & Accessibility**
- [ ] Lighthouse/a11y testing plan in place
- [ ] GA4 and form tracking integration planning
- [ ] SEO schema blocks (Preserves, Stories, Events) scoped

---

## Phase 5: Launch Prep (Q1 2026)

Preparing for a clean, phased go-live with staff onboarding, QA, and soft internal rollout.

**Content Migration**
- [x] Migration planning system structured by file-style hierarchy
- [x] Priority pages and redirects mapped
- [ ] Page-level rewrites and content handoff in progress
- [ ] Asset review, link cleanup, and metadata prep planned

**QA + Feedback**
- [ ] Cross-browser/device testing planned
- [ ] Internal team preview and feedback loop
- [ ] Accessibility and navigation stress testing

**Go-Live & Communications**
- [ ] Internal launch (Board and staff)
- [ ] Public announcement timeline coordinated with rebrand
- [ ] Post-launch monitoring and iteration
